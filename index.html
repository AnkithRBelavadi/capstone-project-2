Projects

INTERACTIVE BOT

Input data obtained from the sensors are stored in its memory (SRAM). A comparison is made against a pre-defined set of voice data (Arduino Library). If the comparison is successful, binary data associated with the pattern match is displayed on the serial monitor and a .wav file is played from the library (SDCard).

MIMICKING BOT

System consists of two Robotic arms, one with servo motors and another with 4 potentiometers. Based on the values of the potentiometer (SRAM), servomotors associated with the robotic arm are made to rotate at a particular angle to match the movements.
